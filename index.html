About Me

I am currently studying Software Development at the IT University of Copenhagen. Over the past months, I've been getting more interested in web development. Specifically front end web design is very interesting to me. I love figuring out how to make a website simple to navigate, while still looking beautiful with a cool design. Map program with path & route instructions

A desktop application to display a loaded map file. Complete with pathfinding functionality, route instructions and address searching.

Technology

The program is developed in Java for a school project. The application uses a certain datafile loaded by the user to display a map. This is done by parsing the datafile with Java and drawing the map and interface using Swing. The program was developed by myself and 4 other students, using the IntelliJ Idea IDE for Java development, and git with GitHub for version control and developing across multiple computers and environments.

Weather App

A simple application to display weather conditions in the requested city.

Technology

The app uses a NodeJS backend to request data from the OpenWeatherMap API. The app searches for the city input by the user, and displays relevant weather information for this city. The app is also powered by the NodeJS web application framework Express. This application was my first venture into NodeJS and JavaScript in general, and it tought me a lot about how frameworks operate and how to communicate with services through an API.
